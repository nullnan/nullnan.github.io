字符串的hash

题目 字符串的hash

就是给你很多字符串，让你统计不同的个数

<set>做法

首先呢，其实不用hash也可以，C++ STL里面的<set>就可以解决

AC Code

然后你就会发现神奇的AC了，毕竟<set>就是维护了一个不重复的集合吗

不过它使用红黑树实现 单次插入复杂度O(logn), 那么总的复杂度就是O(nlogn),

而本题的数据范围是n <= 10000,所以自然就能过。

所以本篇文章就水完了，谢谢大家。

HASH

但等等，我们不是要学习HASH吗？

而且……

还是得好好学

那么hash到底是什么呢，其实hash很常见，平常校验文件、镜像时使用的sha256 、md5等算法都是hash算法，他们都是将不同的数据唯一映射到一个长度固定的空间内。

那么具体是怎么实现的呢?

方法有很多，OI中常用的是进制hash

就是把字符串看做一个base进制的数，每次将hash乘以base然后加上下一个字母ascii值然后对一个数取模,对于模数的选取要尽量大而且尽量是素数，不然就会增加hash对撞的风险（生日悖论），这里一般采用“自然溢出法”，利用unsigned long long的最大值2e64-1是个素数的原理

AC code

然而这是单hash 在很多情况下会被出题人卡掉QAQ

所以可以用双/多hash来防止单个hash的对撞风险。
