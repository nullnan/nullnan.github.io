发布文章

使用

来新建文章，使用markdown编辑器打开source/_posts/文章标题.md即可开始写作。

使用hexo g && hexo s在本地预览这个命令会自动检测文件变动，也就是说，你修改一下，刷新浏览器，就会渲染一次，挺方便的

使用hexo d -g发布到Github Pages，由于网络原因，您可能需要过几分钟才能看到变化。

结尾

由于Github网络技术落后，无法正常链接到中国互联网，请访问https://你的Github用户名.github.io来查看效果 不上HSTS不就是技术落后吗

如果您有自己的私有域名，添加CNAME记录指向你的Blog地址即可，当然您也可以使用智能DNS来进行分流操作。

对于markdown写作软件安利一波Typora，非常舒适
